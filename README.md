# left-recursion-killer

大三 - 编译原理 - 左递归消除程序

修正课本P29页的算法如下：

```c++
把文法G中的所有非终结符号排序为A1, A2, ..., An
for (i = 1; i <= n; ++i){
	消除Ai产生式中的直接左递归
	for (j = 1; j <= i - 1; ++j){
		flag = false; // 是否调用了原算法第5行
		if (Aj -> δ1 | δ2 | ... | δn是当前关于Aj的所有产生式){
			if (存在形如Ai -> Ajγ的生成式){
				把每个形如Ai -> Ajγ的产生式改写为Ai -> δ1γ | δ2γ | ... | δnγ
				flag = true;
			}
			消除Ai产生式中的直接左递归
		}
		if (flag) j = 0; // 重置j
	}
}
化简文法，去除无用的非终结符和产生式
```
